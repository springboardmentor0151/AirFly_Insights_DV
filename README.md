# ✈️ AirFly Insights Dashboard

## 📌 Project Overview
This project analyzes airline operations data to understand patterns into:
- Flight delays
- Airport traffic
- Flight cancellations
- Seasonal trends

The goal is to derive insights and present them through visualizations and an interactive dashboard.


## 🎯 Objectives
- Analyze airline performance data
- Identify key delay factors
- Understand airport traffic patterns
- Visualize trends using graphs
- Build an interactive Streamlit dashboard

## 📊 Key Insights
- Major airports handle high traffic leading to congestion
- Carrier and weather delays are common
- Delay patterns vary across seasons
- Flight cancellations are influenced by operational and environmental factors
